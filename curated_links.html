Curated Links

A collection of valuable resources I've discovered and found useful. I regularly update this list with new finds across various categories.

Tutorials

  • SQL Noir - Solve mysteries through SQL. Step into the shoes of a real detective and solve crimes by writing SQL queries! SQL Noir is an interactive mystery-solving game where you crack cases by writing SQL queries.
  • The Modern JavaScript Tutorial - From the basics to advanced topics with simple, but detailed explanations. This is one of the most comprehensive JavaScript references I've found with practical examples.
  • The Coding Train - Daniel Shiffman's YouTube channel featuring creative coding tutorials on topics like algorithms, machine learning, visualization, and generative art. His enthusiasm makes complex topics accessible.

Tools

  • GeoAI: Artificial Intelligence for Geospatial Data - Provides tools for processing, analyzing, and visualizing geospatial data using advanced ML techniques.
  • Excalidraw - A virtual whiteboard for sketching hand-drawn like diagrams. Perfect for creating system architecture diagrams, flowcharts, and mockups with a unique sketched aesthetic.
  • Figma - Collaborative interface design tool that works in the browser. I use this for all my UI/UX design work, particularly for its powerful prototyping and collaboration features.

Resources

  • Forecasting: Principles and Practice, the Pythonic Way - Online textbook on forecasting for Python.
  • The Non-Technical Guide to Machine Learning - An excellent introduction to machine learning concepts explained in plain language without requiring advanced math or statistics knowledge.
  • Refactoring Guru - Comprehensive resource on software design patterns and refactoring techniques with clean illustrations and practical examples across multiple programming languages.

Books

  • Designing Data-Intensive Applications - Martin Kleppmann's masterpiece on the architecture of data systems. This book changed how I think about building scalable applications.
  • Clean Code - Robert C. Martin's classic on writing code that's easy to understand, refactor, and maintain. Still relevant after all these years.
